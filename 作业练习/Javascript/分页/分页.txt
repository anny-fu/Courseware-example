
封装如下形式的函数：
page({
	id: 分页容器id,
	curPage: 当前页，默认为1,
	allPage: 总页数,
	callBack: function(curPage, allPage) {
	//实时显示当前页和总页数
		var p = document.getElementsByTagName('p')[0];
		p.innerHTML = '当前页：' + curPage + '，总共页：' + allPage;
	}
});

代码思路：
1.【首页】仅在当前页大于等于4，且总页数大于等于6的情况下出现首页
2.显示【上一页】在当前页大于等于2的情况下出现上一页
3.【尾页】当总页数比当前页至少大3，且总页码数大于等于6的情况下出现
4.【下一页】当前页不等于总页码数且总页码数大于等于2的情况下
5.假设页码显示的数量为五个，则分为两种情况
（1）、总页码小于等于5
（2）、总页码大于5
6.为每一个a添加点击事件
7.执行回调函数，实时显示每次点击后的当前页和总页码
